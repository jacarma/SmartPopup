Sandbox to discuss the implementation of SmartPopup control in OpenLayers.

Run

You can run this proposal in:
  • Initial example (to withdraw) with an vector layer: smart-popup.html
  • Multi layer using URI instead of Id: smart-popup2.html (see "templates" folder)
  • Using Internationalized templates: smart-popup-i18n.html, thinking about social layers.
    (see template "sprintCode-i18n.html")

You can also clone the project with Git by running: 

$ git clone git://github.com/jorix/SmartPopup

get the source code on GitHub: jorix/SmartPopup
